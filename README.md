# CarSee.ai Website Clone

This is a complete clone of the CarSee.ai website (https://www.carsee/), built with HTML, CSS, and JavaScript.

## Project Structure

```
.
├── index.html          # Main HTML file
├── css/
│   ├── style.css      # Main stylesheet
│   └── responsive.css # Responsive design styles
├── js/
│   └── main.js        # JavaScript for interactivity
├── assets/
│   ├── images/        # Solution and feature images
│   ├── icons/         # Icon files
│   └── logos/         # Logo files (partners, company)
└── download-assets.ps1 # PowerShell script to download assets

## Features

- **Exact replica** of the original CarSee.ai website
- **Responsive design** that works on all devices
- **All assets downloaded** locally from the original website
- **Interactive navigation** with dropdown menus
- **Mobile-friendly** hamburger menu
- **Smooth animations** and transitions
- **Survey banner** with close functionality

## How to Use

1. Open `index.html` in a web browser
2. All assets are stored locally in the `assets/` folder
3. The website is fully functional and matches the original design

## Assets

All images, logos, and icons have been downloaded from the original website and stored locally. The download script (`download-assets.ps1`) can be run again to update assets if needed.

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Notes

- The website uses local assets, so it works offline
- All styling matches the original website's design
- Responsive breakpoints are set for mobile, tablet, and desktop
